`scala`的基础字面量分为一下几种，参考：`DataType`：

 - **整型字面量**
> 整型字面量用于`Int`类型，如果表示`Long`，可以在数字后面添加`L`或者小写的`l`作为后缀。例如：
> ```
> 0
> 035
> 10L
> 5l
> ```

 - **浮点型字面量**
> 如果浮点数后面有`f`或者`F`，表示这是一个`Float`类型，否则就是`Double`类型。如下：
> ```
> 0.0
> 1.2F
> 3.0f
> ```

 - **布尔型字面量**
> 布尔型字面量有`true`和`false`两种

 - **符号字面量**
> 符号字面量被写成：'<标识符>，这里 <标识符>可以是任何字母或者数字的标识。这种字面量被映射成预定义类`scala.Symbol`的实例。如下：
> `符号字面量`'x`是表达式`scala.Symbol("x")`的简写，符号字面量定义如下：
> ```
> pacakge scala
> 
> final case class Symbol private (name: String) {
>   override def toString: String = "'" + name
> }
> ```

 - **字符字面量**
> 在`scala`中字符字面量用单引号`'`定义，如下：
> ```
> 'a'
> '1'
> '\n'
> ```

 - **字符串字面量**
> 在`scala`中字符串字面量用双引号`"`定义，如下：
> ```
> "Hello Scala!"
> ```
>

 - **多行字符串的表示方法**
> 多行字符串用三个双引号来表示分割，格式为：`"""..."""`，如下：
> ```
> val a = """
> www.widdo.cn
> 这是我的域名
> 在这里，记录学习和生活...
> """
> ```
>

 - **`Null`值**
> 空值是`scala.Null`类型。

 - **`Nothing`类型**
> `Nothing`是所有类型的子类，也是`Null`的子类，含义便是啥也没有，所以没有任何实例。作用：
> 1. 用于参数类型的自动推断
> ```
> List() -> List[Nothing]
> ```
> 2. 可表示非正常退出
> ```
> def e = throw new Exception("error message")
> ```
> 调用`e`将抛出异常，也是啥值都没有，所以就是`Nothing`
